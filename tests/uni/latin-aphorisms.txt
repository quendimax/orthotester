# Лацінскія афарызмы (1 сем)

# 0
Больш слухай, менш гавары -> Audi multa, loquere pauca
Праўда ў віне -> In vino veritas
Пасля віна — словы, як пасля дажджу ўсходзяць травы -> Post vinum verba, post imbrem nascitur herba
Праз церні да зор -> Per aspera ad astra
Зашыць некаму рот -> Consuere alicui os
# 5
Новы кароль —  новы закон -> Novus rex, nova lex
Бойся данайцаў, нат тых, што дарункі прыносяць -> Timeo Danaos et dona ferentes
Прыйшоў, пабачыў, перамог -> Veni, vidi, vici  # Caesar
Там перамога, дзе згода -> Ibi victoria, ubi concordia  # Publius Syrus
Пунійская адданасць -> Punica fides
# 10

Альбо цэзар, альбо нічога -> Aut caesar, aut nihil
Перамога або смерць -> Aut vincere, aut mori
Гораду і свету -> Urbi et orbi
Словы адлятаюць, напісы застаюцца -> Verba volant, scripta manent
Хто ўсяго хоча, усё траціць -> Qui totum vult, totum perit
# 15
Закон ёсць закон -> Dura lex, sed lex
Мыслю, значыць існую -> Cogito, ergo sum
Жыццё без сяброў — самота -> Vita sine amicis tristitia est
Голас народу — голас Божы -> Vox populi — vox dei
Жыць — значыць змагацца -> Vivere militare est  # Seneca
# 20

Праўда па-над усім -> Super omnia veritas
Праўда спараджае нянавіць -> Veritas odium parit
Нараджаюць горы, народзіцца смешнае мышаня -> Parturiunt montes, nascetur ridiculus mus
Адзін чалавек — ніякі чалавек -> Unus vir — nullus vir  # Horatius
Абое рабое (Абое з Аркадыі) -> Arcades ambo  # Vergilius
# 25
Меч раніць цела, слова — душу -> Gladius ferit corpus, animos oratio
Выйсці замуж за сваю жонку -> Uxori nubere suae  # Martialis
Дзе хораша, там і Радзіма -> Ubi bene, ibi patria  # Pacuvius
Або са шчытом, або на шчыце -> Aut cum scuto, aut in scuto
Пакуль дыхаю, спадзяюся -> Dum spiro, spero
# 30

Дрэнны муж — малая бяда, благая жонка — вялікая -> Res mala vir malus est, mala femina pessima res est
Помні, як доўга цягнецца зіма -> Cogitato, quam longa sit hiems
Канец вянчае дзела -> Finis coronat opus
Грош гроша нажывае -> Nummus nummum parit
Ваўкі пабачылі мяне першымі -> Lupi me viderunt priores
# 35
У мутнай вадзе чэрці водзяцца -> Ab aqua silente cave
