# Лацінскія афарызмы (1 сем)

Больш слухай, менш гавары -> Audi multa, loquere pauca
Праўда ў віне -> In vino veritas
Пасля віна — словы, як пасля дажджу ўсходзяць травы -> Post vinum verba, post imbrem nascitur herba
Праз церні да зор -> Per aspera ad astra
Зашыць некаму рот -> Consuere alicui os
Новы кароль —  новы закон -> Novus rex, nova lex
Боюсь данайцев и дары приносящих -> Timeo Danaos et dona ferentes
Прыйшоў, пабачыў, перамог -> Veni, vidi, vici  # Caesar
Там перамога, дзе згода -> Ibi victoria, ubi concordia  # Publius Syrus
Пунійская адданасць -> Punica fides

Альбо цэзар, альбо нічога -> Aut caesar, aut nihil
Перамога або смерць -> Aut vincere, aut mori
Гораду і свету -> Urbi et orbi
Словы адлятаюць, напісы застаюцца -> Verba volant, scripta manent
Хто ўсяго хоча, усё траціць -> Qui totum vult, totum perit
