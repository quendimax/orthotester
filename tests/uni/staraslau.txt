# Спрежэнні дзеяслова БЫТИ

цяп. час, адз. лік -> есмь еси естъ
цяп. час, мн. лік -> есмъ есте сѫтъ
цяп. час, падв. лік -> есвѣ еста есте

аорыст, адз. лік -> быхъ бы(стъ) бы(стъ)
аорыст, мн. лік -> быхомъ бысте бышѧ
аорыст, падв. лік -> быховѣ быста бысте

імперфект, адз. лік -> бѣахъ бѣаше бѣаше
імперфект, мн. лік -> бѣахомъ бѣашете бѣахѫ
імперфект, падв. лік -> бѣаховѣ бѣашета бѣашете  # 3 ас. — беашете

сцяж. імперфект, адз. лік -> бѣхъ бѣ бѣ
сцяж. імперфект, мн. лік -> бѣхомъ бѣсте бѣшѧ
сцяж. імперфект, падв. лік -> бѣховѣ бѣста бѣсте

будучы час, адз. лік -> бѫдѫ бѫдеши бѫдетъ
будучы час, мн. лік -> бѫдемъ бѫдете бѫдѫтъ
будучы час, падв. лік -> бѫдевѣ бѫдета бѫдете

аптатыў, адз. лік -> бимь(бихъ) би би
аптатыў, мн. лік -> бимъ(бихомъ) бисте бѫ(бишѧ)
аптатыў, падв. лік -> биховѣ биста бисте
