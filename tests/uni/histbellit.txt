# Перыядызацыя
Сярэднявечча -> 11-15 стст.
Ранняе сярэднявечча -> 11-13 стст.
Позняе сярэднявечча -> 14-15 стст.
Рэнесанс -> 16 ст.
Барока -> к 16 - пп 18 ст.

Літаратура Кіеўскай Русі -> 11-13 стст.
Літаратура перыяду станаўлення ВКЛ -> 14 - пач 16.
Літаратура Адраджэння і ранняга барока -> дрп 16 - пач 17 ст.
Літаратура пераходнага перыяду -> дрп 17 - 18 ст.

легендарна-рэлігійныя творы, блізкія да кананічных тэкстаў, але з іншай трактоўкай падзей -> апокрыфы
раннехрысціянскія творы айцоў царквы -> патрыстыка
жыційная літаратура -> агіяграфія
Першасны, выточны тэкст -> пратограф
Якасная копія пратографа (без зменаў, што ўплываюць на сюжэт, і аўт. дадаткаў) -> спіс
Тэкст створаны на аснове некалькіх спісаў -> рэдакцыя
Перапісаны тэкст з мясцовымі моўнымі асаблівасцямі -> ізвод

Кнігі, каторыя тлумачаць тэксты Бібліі -> палеі
Год напісання АМГ подле Шахматава -> 1095
Перапрацаванне на загад Уладзіміра Манамаха -> 1116
Перапрацаванне на загад князя Мсціслава -> 1118

Год стварэння Лаўрэнцьеўскага летапіса -> 1377
Год стварэння Супрасльскага летапіса -> 1446

Іларыён быў пасвечаны саборам епіскапаў у мітрапаліты -> 1051
Тры часткі слова -> эксордыум, нарацыя, канклюзія
