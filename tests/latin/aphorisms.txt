# Лацінскія афарызмы (1 сем і 2 сем)

# 0
Больш слухай, менш гавары -> Audi multa, loquere pauca
Праўда ў віне -> In vino veritas
Пасля віна — словы, як пасля дажджу ўсходзяць травы -> Post vinum verba, post imbrem nascitur herba
Праз церні да зор -> Per aspera ad astra
Зашыць некаму рот -> Consuere alicui os
# 5
Новы кароль —  новы закон -> Novus rex, nova lex
Бойся данайцаў, нат тых, што дарункі прыносяць -> Timeo Danaos et dona ferentes
Прыйшоў, пабачыў, перамог -> Veni, vidi, vici  # Caesar
Там перамога, дзе згода -> Ibi victoria, ubi concordia  # Publius Syrus
Пунійская адданасць -> Punica fides
# 10

Альбо цэзар, альбо нічога -> Aut caesar, aut nihil
Перамога або смерць -> Aut vincere, aut mori
Гораду і свету -> Urbi et orbi
Словы адлятаюць, напісы застаюцца -> Verba volant, scripta manent
Хто ўсяго хоча, усё траціць -> Qui totum vult, totum perit
# 15
Закон ёсць закон -> Dura lex, sed lex
Мыслю, значыць існую -> Cogito, ergo sum
Жыццё без сяброў — самота -> Vita sine amicis tristitia est
Голас народу — голас Божы -> Vox populi — vox dei
Жыць — значыць змагацца -> Vivere militare est  # Seneca
# 20

Праўда па-над усім -> Super omnia veritas
Праўда спараджае нянавіць -> Veritas odium parit
Нараджаюць горы, народзіцца смешнае мышаня -> Parturiunt montes, nascetur ridiculus mus
Адзін чалавек — ніякі чалавек -> Unus vir — nullus vir  # Horatius
Абое рабое (Абое з Аркадыі) -> Arcades ambo  # Vergilius
# 25
Меч раніць цела, слова — душу -> Gladius ferit corpus, animos oratio
Выйсці замуж за сваю жонку -> Uxori nubere suae  # Martialis
Дзе хораша, там і Радзіма -> Ubi bene, ibi patria  # Pacuvius
Або са шчытом, або на шчыце -> Aut cum scuto, aut in scuto
Пакуль дыхаю, спадзяюся -> Dum spiro, spero
# 30

Дрэнны муж — малая бяда, благая жонка — вялікая -> Res mala vir malus est, mala femina pessima res est
Помні, як доўга цягнецца зіма -> Cogitato, quam longa sit hiems
Канец вянчае дзела -> Finis coronat opus
Грош гроша нажывае -> Nummus nummum parit
Ваўкі пабачылі мяне першымі -> Lupi me viderunt priores
# 35
Сцеражыся ціхай вады -> Ab aqua silente cave
Мыць эфіёпа -> Aethiopem lavare
Помні, што ты чалавек -> Te hominem esse memento
Што маем, не шануем -> Quod habet, non numerat
Без барады, а старых вучыш -> Ante barbam doces senes
# 40

Цешся жыццём, пакуль жывеш -> Fruere vita, dum vivis
Зрабіць падушку з меча -> Gladium facere culcitam
Помні пра смерць -> Memento mori
Здаравейшы за рыбу -> Sanior est pisce
Колькі людзей, столькі думак -> Quot homines, tot sententiae
# 45
Даручаць ваўку барана -> Ovem lupo committere
Запас не нястача -> Copia non est inopia
Чый край, таго і мова -> Cujus regio, ejus lingua
Пад гэтым знакам пераможаш -> In hoc signo vinces
Цярпенне ўсё перамагае -> Patientia omnia vincit
# 50

Аўрора — Музам сяброўка -> Aurora Musis amica est
Майстрам робяцца ў справе -> Fabricando fit faber
Хто пʼе віно, няхай пʼе і асадак -> Faecem bibat, qui vinum bibit
Гультаю заўсёды свята -> Ignavis semper feriae    # Erasmus Roterodamus
Нястомная праца ўсё перамагае -> Labor omnia vincit improbus
# 55
Хто не працуе, той не есць -> Qui non laborat, non manducet
З дзвюх бед выбірай меншую -> E deobus malis minus est eligendum
Падліваць масла ў агонь -> Oleum addere camino
Упаў у яму, каторую выкапаў іншаму -> Incidit in foveam, quam fecit alteri
Сабе дабро робіць той, хто робіць дабро другу -> Sibi bene facit, qui bene facit amico
# 60

Розум проці сілы мала значыць -> Ratio contra vim parum valet
Кахаць і заставацца разумным ледзьве богу здараецца -> Amare et sapere vix deo conceditur  # Publius Syrus
Свой дом — лепшы дом -> Domus propria — domus optima
Двойчы дае той, хто дае скора ->Bis dat, qui cito dat
Шчасце мае многа сяброў -> Felicitas multos habet amicos
# 65
Шмат рук аблягчаюць цяжар -> Multae manus onus levant
Высокае крэсла не прыме дваіх -> Summa sedes non capit duos
Чым болей павагі, тым большы цяжар -> Quo plus honoris, eo plus oneris
Мёртвага льва нават шчаняты кусаюць -> Leonem mortuum etiam et catuli mordent
Размясціць усё на адным караблі -> Uni navi committere omnia
# 70

Доля рачыць дурням -> Fortuna favet fatuis
Нешчаслівы, хто ніколі не быў нешчаслівым -> Miser, qui nunquam miser
Чаго не бачыць вока, ад таго сэрца не баліць -> Quod non videt oculus, cor non dolet
Навучаны бядой -> Calamitate doctus sum
Пазычаць душу -> Animum debere
# 75
Голад — вучыцель рамёстваў -> Fames artium magistra
Грошы не пахнуць -> Pecunia non olet
Галодны натоўп не знае страху -> Nescit plebs jejuna timere
Купляць ката ў мяшку -> Emere catulum in sacco
Дае курыцу, каб атрымаць вала -> Gallinam dat, ut taurum recipiat

# 80
Рука руку мые -> Manus manum lavat
Па смерці лячэнне -> Post mortem medicina
Глухому казку апавядаеш -> Surdo fabellam narras
Ганарыстасць робіць дурнем -> Arrogantia facit stultum
Хто за двума зайцамі пагоніцца, ніводнага не зловіць -> Duos qui lepores sequitur, neutrum capit
# 85
Увесь із шкатулкі -> Totus de capsula
Мёд на вуснах, малако на словах, жоўць на сэрцы, фальш у справах -> Mel in ore, verba lactis, fel in corde, fraus in factis
Пра густы не спрачаюцца -> De gustibus non est disputandum
Вялікая душа вялікага шчасця варта -> Felicitatem ingentem animus ingens decet
Не кожны можа быць патрыцыем -> Non possunt omnes esse patricii

# 90
Што можна Юпітэру, не можна быку -> Quod licet Jovi, non licet bovi
Свая радзіма кожнаму найлепшая -> Sua cuique patria jucundissima est
Дзень дня вучыць -> Dies diem docet
На векі векаў -> In saecula saeculorum
Жыві мерна, як сродкі дазваляюць -> Vivas frugalis, ut res est materialis
# 95
Я ведаю, што нічога не ведаю -> Scio me nihil scire     # Socrates
Аратарамі робяцца, паэтамі нараджаюцца -> Oratores fiunt, poetae nascuntur
Ніхто не суддзя ў уласнай справе -> Nemo judex in propria causa
Рыба спачатку з галавы гніе -> Piscis primum capite foetet
Ведаеш, кот, каму ліжаш бараду і каму пяеш курну -> Scis, Catte, cui lingis barbam, cui murmura fingis
